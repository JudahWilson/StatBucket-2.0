# NBA Basketball Reference Scraper

A comprehensive system for scraping NBA data from basketball-reference.com.

## Installation

````bash
# Install dependencies
uv sync

# Initialize development environment
uv run dev-setup```

## Commands

- To see all tertiary scripts, run:

```bash
uv run list-scripts
````
